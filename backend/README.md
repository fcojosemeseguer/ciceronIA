# Ciceron - Backend API

## Overview
This repository contains the backend service and API wrappers for the project. 

## Tech Stack
- Backend: Python
- Framework: FastAPI
- Integration: OpenAI API, OpenSmile, Whisper

## Setup Instructions

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run:
    ```bash
    uvicorn main:app --reload
    ```
