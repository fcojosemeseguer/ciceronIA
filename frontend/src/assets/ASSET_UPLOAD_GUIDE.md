# Asset Upload Guide (Figma -> Frontend)

Sube aqui todos los recursos del nuevo rediseño:

## 1) Fuentes

Ruta:
- `frontend/src/assets/fonts/`

Archivos esperados:
- `DalekPinpoint-Regular.woff2` (obligatorio)
- `DalekPinpoint-Regular.woff` (opcional fallback)

Si tienes variantes:
- `DalekPinpoint-Bold.woff2`
- `DalekPinpoint-Light.woff2`

## 2) SVG de interfaz

Ruta:
- `frontend/src/assets/icons/`

Nombres sugeridos:
- `logo-ciceronia.svg`
- `logo-laurel-left.svg`
- `logo-laurel-right.svg`
- `icon-settings.svg`
- `icon-back.svg`
- `icon-live.svg`
- `icon-audio-analysis.svg`
- `icon-dashboard.svg`
- `icon-history.svg`
- `icon-play.svg`
- `icon-pause.svg`

## 3) Ilustraciones/elementos grandes

Ruta:
- `frontend/src/assets/illustrations/`

Nombres sugeridos:
- `panel-empty-state.svg`
- `debate-live-header.svg`
- `analysis-empty-state.svg`

## 4) Export de tokens desde Figma (muy recomendado)

Sube un JSON con:
- Tipografias
- Colores
- Espaciados
- Radios
- Bordes
- Sombras

Ruta:
- `frontend/src/assets/tokens/figma-tokens.json`

Si no tienes JSON, puedes subir un `.txt` con valores.
